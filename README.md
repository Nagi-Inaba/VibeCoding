# Vibe Coding Guide

バイブコーディング初心者向けの複数ページガイドサイトです。

短い指示でもAIが動きやすくなるように、作業場所の整え方、場面別の頼み方、フォルダ構成、作業記録の残し方をまとめています。

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

`main` ブランチへpushすると、Vercelへ自動デプロイされます。

公開URL: https://vibecoding-jp.vercel.app
